# Student Engagement Intelligence System

## Overview
This project analyzes student engagement data from PragyanAI 
to predict placement outcomes and identify key behavioral 
patterns that drive student success.

## Dataset
50,000 student records with 19 features covering attendance, 
login activity, video completion, quiz performance, doubt 
interaction and event participation.

## What This Project Does
- Exploratory Data Analysis (EDA) with visualizations
- Feature Engineering — created Engagement Score, 
  Learning Effectiveness and Interaction Score
- Machine Learning models — Logistic Regression, 
  Random Forest and XGBoost
- Student Segmentation — High Performers, Passive Learners, 
  Active but Confused, Disengaged
- Correlation Analysis
- Business Recommendations for PragyanAI

## Key Findings
- 60% placement rate overall
- CGPA and Learning Effectiveness are top predictors
- 41% of students are disengaged — urgent intervention needed
- Only 18% are High Performers ready for placement

## Technologies Used
- Python, Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn, XGBoost
